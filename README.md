
# EJERCICIO DAW

## Descripcion
Esta aplicacion lo que hace es conectarse a una base de datos y a un apache mediante docker con lo cual podemos crear un listado

## Requisitos
    - docker
    - git
## Guia de instalación
    1. Hacemos un git clone del siguiente repositorio
        -git@github.com:AlejandroSucilla/MANU-COMPOSER.git
2. Despues de hacer el clone entramos en la carpeta del repositorio
    3. Abrimos el terminal dentro de esta carpeta (navegas con la terminal o dentro de la carpeta click derecho y abrir la terminal)
    4. Por utlimo hacemos el comando " docker-compose up -d "
## DIRECCION URL
    - localhost:83
## ARQUITECTURAS
    - Sericio WEB
        - Apache con PHP
    - Servicio BBDD
        - MYSQL
